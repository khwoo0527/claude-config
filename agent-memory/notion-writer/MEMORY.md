# Notion Writer — 작업 컨텍스트 캐시

> 이 파일의 용도/기록 시점/형식은 [`agent-memory 정책`](../../rules/workflow/agent-memory.md) 을 따른다.
>
> **담는 것**: 이 프로젝트의 Notion 페이지 ID, 스타일 피드백, API 특이사항, 페이지별 업데이트 이력 등 다음 호출 시 빠른 컨텍스트 복원용.
> **담지 않는 것**:
> - 다른 프로젝트에 통하는 Notion 작성 노하우 → `rules/workflow/notion.md` 또는 `agents/notion-writer.md` 본문
> - 사용자 작업 스타일 → `memory/feedback_*.md`
> - Integration 토큰/페이지 ID 같은 민감 정보 → `CLAUDE.md` (프로젝트 정의서) 또는 `.env`
